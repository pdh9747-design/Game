# 점프맵 게임

## 게임 설명
플레이어가 점프를 이용하여 장애물을 통과하고 모든 아이템을 수집하는 게임입니다.

## 조작법
- W : 전진
- A : 왼쪽 이동
- S : 후진
- D : 오른쪽 이동
- Space : 점프

## 구현 내용

### 플레이어 이동
Input System을 사용하여 키보드 입력을 처리하였습니다.
Vector3와 normalized를 사용하여 이동 방향을 계산하였습니다.

### 몬스터 감지 및 회전
Vector3.Dot을 사용하여 시야각 판정을 구현하였습니다.
Quaternion.Slerp를 사용하여 장애물을 향해 부드럽게 회전하도록 구현하였습니다.

### 거리 판정
sqrMagnitude를 사용하여 거리 판정을 구현하였습니다.

### 아이템 관리
List<GameObject>를 사용하여 아이템을 관리하였습니다.
아이템 획득 시 List에서 제거되며 점수가 증가합니다.

### 길찾기
Queue<Transform>을 사용하여 노드를 순서대로 방문하는 경로 이동을 구현하였습니다.
노드(Node)를 이용하여 이동 경로를 관리합니다.
